# dnsight

**dnsight** is a Python SDK and CLI for auditing DNS, email authentication (SPF, DKIM, DMARC), and related signals.

## Quick links

- [CLI overview](cli/index.md) — command tree, global options, generated reference
- [Configuration](configuration.md) — `dnsight.yaml` (v1)
- [API overview](api/index.md) — Python package reference
- [Contributing](contributing.md) — how to work on the repo
- [dnsight on PyPI](https://pypi.org/project/dnsight/) — releases and `pip install`

## Install

```bash
pip install dnsight
```

See the [README on GitHub](https://github.com/dnsight/dnsight/blob/main/README.md) for the full quickstart, logging, and output formats.
